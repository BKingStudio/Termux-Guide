📟 Termux Commands Reference

A complete CLI guide for Android + Linux users

🧰 Chapter 1: Basic Commands

📦 Package Management

# Command Description Action
1 pkg update Updates the package lists for upgrades and installs.
2 pkg upgrade Upgrades installed packages to the latest version.
3 pkg install [package_name] Installs a specified package.
4 pkg uninstall [package_name] Uninstalls a specified package.
5 pkg search [keyword] Searches packages matching the keyword.
6 pkg info [package_name] Shows info about a specific package.
7 pkg list-all List all available packages.
8 pkg list-installed Lists all installed packages.

📂 Storage & File Access

# Command Description Action
1 termux-setup-storage Grants Termux access to device storage.
2 termux-open [file/path] Opens a file or directory.
3 termux-download [URL] Downloads a file from a URL.
4 termux-upload [source] [destination] Uploads a file.
5 termux-storage-get [path] Gets storage info of a path.
6 termux-storage-list Lists all storage volumes.

🔒 Device Control & Status

# Command Description Action
1 termux-wake-lock Keeps the device awake.
2 termux-wake-unlock Unlocks the device sleep mode.
3 termux-battery-status Shows battery status.
4 termux-info Displays device/system info.
5 termux-wifi-enable [true/false] Enables/disables Wi-Fi.
6 termux-wifi-connectioninfo Shows current Wi-Fi connection info.
7 termux-wifi-ipinfo Shows current Wi-Fi connection info.
8 termux-wifi-scaninfo Scans available Wi-Fi networks.
9 termux-telephony-cellinfo Cellular network info.
10 termux-telephony-deviceinfo Device info.
11 termux-telephony-imei IMEI number.
12 termux-telephony-signalstrength Cellular signal strength.

🔈 Audio, Vibration & Media

# Command Description Action
1 termux-volume [option] Adjusts volume settings.
2 termux-vibrate Vibrates the device.
3 termux-media-player play [file] Plays media file.
4 termux-media-player pause Pauses media.
5 termux-media-player stop termux-media-player stop
6 termux-media-player seek [time] Seeks to specific time.
7 termux-media-player playlist [file] Loads a playlist.
8 termux-media-scan [path] Scans media files in a path.
9 termux-microphone-record [output_file] Records audio from mic.
